# 🚀 GitHub Pages Pe Deploy Karne Ka 3-Minute Guide

## Sirf 4 Steps:

### 1️⃣ GitHub Repo Banao
- https://github.com/new pe jao
- Name: `proptrade-demo`
- ✅ Public select karo
- "Create repository" click

### 2️⃣ Files Upload Karo
- Repo page pe **"uploading an existing file"** link click
- `proptrade-web/` folder ke saare files drag-drop karo
- ⚠️ **`.nojekyll`** file bhi include karo (hidden file — show hidden enable karo)
- Niche **"Commit changes"** click

### 3️⃣ GitHub Pages Enable Karo
- Repo ka **Settings** tab kholo
- Left sidebar: **Pages**
- Source: Branch = **`main`**, Folder = **`/ (root)`**
- **Save** click

### 4️⃣ 1-2 minute baad URL milega:
```
https://YOUR_USERNAME.github.io/proptrade-demo/
```

✅ Yahi URL client ko share karo.
